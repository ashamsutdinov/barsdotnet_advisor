# barsdotnet_advisor

2. Сайт рекомендаций
Реализовать web-приложение для сбора оценок пользователей на разного вида товары/услуги
Требования:
* система регистрации/авторизации
* возможность добавлять предмет оценки (товар/услуга/работадатель - уточнить справочник видов предеметов оценки)
* придумать или подсмотреть систему оценок (тут есть варианты: плюсики, баллы. разные способы подсчёта скользящего среднего с учётом авторитета голосующего или без)
* возможность добавлять комментарии к предмету
* главная страница с галереей более-менее оценённых предметов плюс новые предметы для оценки
 
2.2 можно делать либо обобщённый вариант, который указан в пункте 2, либо конкретизировать предмет оценки (товары, фотографии, гостиницы и пр.)
