# barsdotnet_aggregator

Агрегатор микроблогов
Реализовать web-приложение для отслеживания множества микроблогов (для начала только твиттеры).
Требования:
* система регистрации, авторизации
* возможность пользователю добавлять микроблог (пирмер формата добавления https://twitter.com/barsgroup)
* лента сообщений для пользователя из его добавленных микроблогов
